How to Add to Your Website

Step 1: Add HTML Placeholder

Place this span element where you want the Nepali date to appear:

<span id="mypatro_nepali_date"></span>

Step 2: Add Script Tag

Add this script before the closing </body> tag:

<script src="https://www.ashesh.com.np/linknepali-time.php?format=1"></script>

Format Options

format=1 - Full Format with Day
format=2 - Date Only
format=3 - Short Format
format=4 - Custom Format

Complete Example

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>Today's Nepali Date: <span id="mypatro_nepali_date"></span></p>
    
    <script src="https://www.ashesh.com.np/linknepali-time.php?format=1"></script>
</body>
</html>
